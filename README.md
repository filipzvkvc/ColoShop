# ColoShop 🛍️

A full-stack e-commerce platform for managing and selling shoes, clothing, and accessories.

## 🌐 Live Demo
[coloshop.infinityfreeapp.com](https://coloshop.infinityfreeapp.com/index.php)

## ✨ Features
- User authentication (register/login)
- Product browsing with categories
- Wishlist
- Product reviews
- Shopping cart (guest & logged in)
- Order history
- Newsletter subscription
- Secure checkout with PayPal

## 🛠️ Built With
- PHP
- MySQL
- JavaScript / jQuery / AJAX
- HTML / CSS / Bootstrap 5
